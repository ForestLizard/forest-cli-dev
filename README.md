# 前端脚手架

前端项目脚手架demo，旨在提高前端研发效能，统一技术栈，提升团对工程化水平，保留流程的安装稳定性.
项目采用mono-repo架构，动态加载commmand
初步计划提供以下功能：
1、项目初始化（已完成）；
2、组件初始化（已完成）；
3、git-flow（已完成）；
4、云构建和云发布（开发中）等功能。

## 安装

npm install @forest-cli-dev/core -g

## 使用

forest-cli init 
项目/组件初始化
![forest-cli init](/asserts/init@2x.png)

forest-cli publish
执行git-flow流程
![forest-cli publish](/asserts/publish@2x.png)

forest-cli -h
查看帮助文档
